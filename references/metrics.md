# 标注质量指标

## 核心指标

### 1. 标注准确率 (Annotation Accuracy)

```
准确率 = 正确标注的样本数 / 总样本数 × 100%
```

**分级标准：**
- 🟢 优秀：> 95%
- 🟡 良好：90% - 95%
- 🟠 需改进：80% - 90%
- 🔴 差：< 80%

### 2. 幻觉标注率 (Hallucination Rate)

```
幻觉率 = 含幻觉标注的样本数 / 总样本数 × 100%
```

**这是最严重的错误类型**，需要优先解决。

### 3. 要素级 F1 分数

按要素类型分别计算：

```
Precision = 正确抽取的要素数 / 标注的要素总数
Recall = 正确抽取的要素数 / 实际应标注的要素数
F1 = 2 × Precision × Recall / (Precision + Recall)
```

### 4. 标注者间一致性 (如果有多个标注者)

**Cohen's Kappa** (两人标注):
```
κ = (p₀ - pₑ) / (1 - pₑ)
```

**Fleiss' Kappa** (多人标注):
- κ > 0.8: 几乎完全一致
- 0.6 - 0.8: 高度一致
- 0.4 - 0.6: 中度一致
- < 0.4: 一致性差

## 错误严重程度评分

本技能使用的严重程度算法：

```python
severity = 0
for error in errors:
    if error.type == "hallucination":
        severity += 10    # 最严重
    elif error.type == "missing_text":
        severity += 8
    elif error.type == "invalid_format":
        severity += 5
    else:
        severity += 3

severity += len(warnings) * 0.5
```

## 质量阈值建议

| 场景 | 目标准确率 | 可接受幻觉率 |
|------|-----------|-------------|
| 模型训练数据 | > 98% | < 1% |
| 评估测试集 | > 99% | 0% |
| 快速迭代 | > 95% | < 3% |
| 初版标注 | > 90% | < 5% |

## 监控建议

1. **每日抽检**：随机抽取 50-100 条标注进行验证
2. **错误趋势**：跟踪错误率变化，发现恶化及时干预
3. **要素分布**：监控各要素类型的错误率，针对性优化
